## 2 启动服务
    2.1、docker pull rabbitmq:management
    2.2、docker run -d -p 5672:5672 -p 15672:15672 -e RABBITMQ_DEFAULT_USER=admin -e RABBITMQ_DEFAULT_PASS=secret rabbitmq:management
    2.3、http://localhost:15672/
